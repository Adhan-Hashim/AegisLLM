## [Unreleased]

### Added
- **Landing Page**: New problem-focused landing page with interactive architecture diagram.
- **API Explorer**: Lightweight reference for the REST and Streaming endpoints.
- **Diagnostics Panel**: Developer-mode analytics for UI performance tracking.
- **Session Catalog**: Centralized index for searching and filtering millions of intercepted sessions.
- **Session Viewer**: Unified forensic workspace containing a Timeline, Explainability Graph, and Metrics inspector.
- **Rules Studio**: Declarative YAML editor for crafting deterministic and AI-driven guardrails.
- **Attack Lab**: Interactive interface to simulate malicious prompts against the live pipeline.
- **Streaming Pipeline**: Server-Sent Events (SSE) integration for low-latency feedback.
- **Risk Engine**: Core evaluation engine combining parsers, validators, and executors.

### Changed
- Shifted from a page-oriented UI architecture to a reusable domain-first architecture (Session Viewer).
