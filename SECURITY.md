# Security Policy

## Supported Versions

Currently, we provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within AegisLLM, please do not disclose it publicly. 

Instead, please send an email to the security team at **security@aegisllm.com** (mock email for template). 

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. Once the issue is resolved, we will publish a security advisory and notify the community.
